.. _configuration:

=============
Configuration
=============

An overview of the available extensions and configured feeds can be found in the
:guilabel:`System > Configuration` module.

.. note::
   The :guilabel:`System > Configuration` module is available when the
   :doc:`lowlevel system extension <ext_lowlevel:Index>` is installed.


.. _configuration-extensions:

List of available extensions
============================

In the upper menu bar of the :guilabel:`Configuration` module, select
:guilabel:`Feed Generator: Extensions`.

.. figure:: /Images/ConfigurationModuleExtensions.png
   :alt: Available extensions in the Configuration module

   Available extensions in the :guilabel:`Configuration` module

You can quickly look up an extension using the search box.


.. _configuration-feeds:

List of configured feeds
========================

In the upper menu bar of the :guilabel:`Configuration` module, select
:guilabel:`Feed Generator: Feeds`.

.. figure:: /Images/ConfigurationModuleFeeds.png
   :alt: Configured feeds in the Configuration module

   Configured feeds in the :guilabel:`Configuration` module

The feeds are grouped by the fully-qualified name of the class that implements a
feed.

You can quickly look up a feed using the search box.
